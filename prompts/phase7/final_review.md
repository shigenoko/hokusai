## 出力形式

以下の形式で各ルールのチェック結果を報告してください：

```
## ルール別チェック結果

| ルールID | ルール名 | 結果 | 備考 |
|----------|----------|------|------|
| HQ01 | デッドコードと不要なインポート | OK | - |
| SA01 | サーバーサイド認可 | NG | 権限チェック漏れ |
| UX01 | アクセシビリティと操作性 | SKIP | 該当コードなし |
```

**結果の種類:**
- OK: ルールに適合
- NG: ルール違反を検出
- SKIP: 該当するコードがない

**重要:** 必ず上記の表形式で全てのルール（HQ01〜UX02、およびプロジェクト固有ルールP01〜）の結果を出力してください。
